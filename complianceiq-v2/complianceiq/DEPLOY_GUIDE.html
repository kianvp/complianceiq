1
Upload code to GitHub
GitHub is where your code lives — Render will read it from here
5 min
ℹ️
GitHub is a free website that stores code. Think of it like Google Drive but for code files. You need an account — it's free and takes 1 minute.
1
Go to github.com and click Sign Up. Create a free account.
2
Once logged in, click the + icon (top right) → New repository
github.com/new
REPOSITORY NAME
complianceiq
VISIBILITY
● Public
Create repository
3
Name it complianceiq, set to Public, click Create repository
4
On the next page, click uploading an existing file (the link in the middle of the page)
5
From the complianceiq.zip you downloaded, extract it. Then drag the entire server/ folder AND the render.yaml file into the GitHub upload area.
⚠️
Make sure you upload the server/ folder contents (index.js and package.json) plus render.yaml. The frontend HTML is separate — we'll handle it in Step 4.
6
Scroll down, click Commit changes. Your code is now on GitHub!
2
Deploy on Render.com
Render will read your GitHub code and run it as a live server
5 min
✅
Render's free tier is genuinely free — no credit card needed. Your server runs 24/7.
1
Go to render.com → click Get Started for Free → sign up with your GitHub account (click "Continue with GitHub" — easiest option)
2
In the Render dashboard, click New + → Web Service
3
Click Connect account under GitHub, then select your complianceiq repo
4
Fill in the settings exactly like this:
dashboard.render.com/new/web
NAME
complianceiq-api
ROOT DIRECTORY
server
BUILD COMMAND
npm install
START COMMAND
npm start
INSTANCE TYPE
Free
Create Web Service
5
Click Create Web Service. Render will now build and deploy — takes about 2-3 minutes. You'll see logs scrolling.
6
When you see "Your service is live 🎉" — copy the URL at the top. It looks like: https://complianceiq-api.onrender.com
ℹ️
Test it: paste your URL + /api/feed in a browser. You should see JSON data from RBI and SEBI. If you see it — your server is live!
3
Connect the frontend to your server
One line to change in the HTML file
2 min
1
Open frontend/index.html from your zip in any text editor (Notepad, VS Code, anything)
2
Find this line near the bottom (use Ctrl+F to search for API_URL):
const API_URL = 'http://localhost:3001';
3
Replace http://localhost:3001 with your Render URL:
// Change it to look like this (use YOUR actual Render URL):
const API_URL = 'https://complianceiq-api.onrender.com';
4
Save the file. Open it in your browser — the green LIVE dot should appear in the top right with real RBI/SEBI data!
4
Get a shareable link (Netlify)
Host your frontend so anyone can open it — free, takes 1 minute
2 min
1
Go to netlify.com → Sign up free (use your GitHub account)
2
On the dashboard, look for the "Deploy manually" section — there's a drag-and-drop box that says "Drag and drop your site folder here"
3
Drag your frontend/ folder (the one containing index.html) into that box
4
Netlify instantly gives you a URL like https://magical-unicorn-123.netlify.app — that's your live link! You can share it with your dad.
🎉
You can also rename it to something nicer like complianceiq.netlify.app in Netlify's settings → Site configuration → Change site name.
5
You're live 🎉
Share the link with your dad
Done!
